# CUHKSZ课程群查找器

## 介绍

* warin的首个练手全栈项目
* 前端Vue，后端Express，暂未使用数据库

## 运行

* 运行前端：`cd frontend && npm start`
* 运行后端：`cd server && npm start`
* 项目将会运行在："http://localhost:8080"

## 使用

* "CUHKSZ查找器"页面
  * 输入课程科目和课程编号来筛选课程，获取课程群二维码
  * 点击"添加新课程群"前往"添加新课程群"页面
* "添加新课程群"页面
  * 输入课程科目和课程编号并上传课程群二维码，添加课程群
  * 点击"返回"前往"CUHKSZ查找器"页面
